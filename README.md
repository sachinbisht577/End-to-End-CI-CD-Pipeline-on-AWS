# End-to-End-CI-CD-Pipeline-on-AWS

# 🚀 DevOps CI/CD Project

This project demonstrates a complete CI/CD pipeline using modern DevOps tools.

## 🔧 Tech Stack
- GitHub Actions (CI)
- Docker
- Kubernetes (Helm)
- Terraform (Infrastructure as Code)
- ArgoCD (CD)

## 🚀 Workflow
1. Code pushed to GitHub
2. CI builds Docker image
3. Helm deploys app to Kubernetes
4. ArgoCD manages continuous delivery

## 📦 Project Structure
- app → Node.js app
- terraform → Infrastructure setup
- helm-chart → Kubernetes deployment
- argocd → CD configuration
